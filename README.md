# Uriel

Uriel is a streaming app for watching anime, movies, and series with a fast, modern interface and flexible playback controls.

## Features

- Browse trending anime, movies, and TV series
- Global search across multiple media types
- Rich details pages with episode lists and source selection
- Multiple video players (`ExoPlayer` and `LibMpv`)
- Advanced playback settings (speed, seek interval, resize mode, gestures)
- Subtitle customization (size, colors, outline, background)
- Continue watching and watch history support
- Offline/local persistence support
- Backup and restore support for app data
- Theme system with light/dark/system modes and dynamic color support
